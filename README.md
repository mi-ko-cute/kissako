## Overview
現在地もしくは住所を検索して、そこから近い空いている喫茶店を調べることができる。

## Description
現在地から検索するもしくは住所から検索するボタンをクリックし、そこから半径何ｍの範囲の喫茶店を調べるか指定し検索開始ボタンをクリックする。

検索開始ボタンをクリックすると、GoogleMap上にアイコンで喫茶店の位置が表示される。

またGoogleMapの下に表が存在し、喫茶店の情報が空いている順で表示される。
